# Code of Conduct

trenchkit is a crypto tool, but this is still a collaborative project. A few ground rules keep it functional.

## Expected behavior

- Assume good intent. Read the other person's argument before replying.
- Keep feedback technical. Attack the code, not the coder.
- Say "I don't know" when you don't know. Pretending wastes everyone's time.
- If you disagree with a scorer weight, a design decision, or a naming choice, say so with a reason. "I disagree" alone is not feedback.

## Not acceptable

- Personal attacks, slurs, harassment (public or private).
- Deliberately misleading claims about the code, the tokens it surfaces, or the GMGN API.
- Posting shilled token addresses in issues, PRs, or discussions. This is a tool, not a promotion channel.
- Sharing API keys, private keys, or wallet addresses belonging to other people.
- Opening security issues publicly. See [SECURITY.md](SECURITY.md).

## Enforcement

Maintainers can edit, remove, or reject comments, commits, code, issues, and PRs that violate this code. Repeated or severe violations earn a ban.

Report problems to lw.arts.designer@gmail.com. Reports are kept confidential.

## Scope

This code applies in all project spaces: the repo, issues, PRs, discussions, and any public communication referencing the project.
